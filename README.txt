Running this programm you should automate the reporting process in case you want to track goods turnover of each store in your network 
and visualize the data in percents in a specified timeframe.

Supposed that you have UiPath Studio installed.

Robot fetches the data from Input.xlsm column "A", column "B", "C" . After it forms a report in Report.xlsx where dates from the initial
file are in the header of a reporting file and store codes in the first column.
Robot represents preliminary data and tranforms it in percents as well as coloring cells in red where good's turnover is less than 97%.
